发表于 2017-01-13 | 分类于 数据库 |

mongo安装

CentOS7安装mongdb：
https://docs.mongodb.com/master/tutorial/install-mongodb-on-red-hat/

总共试过3种方法

  1. 官方镜像源，网速太慢，下载不下来
  2. 清华镜像源，无法使用
  3. aliyun的rpm包安装，安装成功 官方镜像源使用过程:

1、创建文件/etc/yum.repos.d/mongodb-org-3.4.repo

文件内容为

[mongodb-org-3.4]
name=MongoDB Repository
baseurl=https://repo.mongodb.org/yum/amazon/2013.03/mongodb-org/3.4/x86_64/
gpgcheck=1
enabled=1
gpgkey=https://www.mongodb.org/static/pgp/server-3.4.asc

使用yum进行安装

yum makecache
yum install mongodb-org

这种方法，网速实在太慢，导致不成功，因此采用aliyun的rpm包安装
安装过程

2、rpm包安装

下载rpm包，总共4个rpm包，选择版本为3.4，包括mongodb-org-mongos、mongodb-org-server、mongodb-org-shell、mongodb-org-tools、mongodb-org

3、安装rpm包，包直接有依赖，需要安装顺序安装

4、修改mongol配置

mongo权限设置

mongodb默认是没有进行账户设置的，最近出现了很多mongodb由于没有设置密码之类的被攻破,因此需要对mongo进行安全的设置 .

没有进行任何安全设置之前，登录的时候会出现大量的warning

安装设置包括：

1、创建用户管理员(超级账户)

第一个用户设置成超级用户，可用于管理其他用户

创建成功

2、端口和ip设置，修改mongo的默认端口，有27017—>xxxxx，然后增加一个绑定指定的ip，使可以进行远程的访问

3、启权限验证

必须开启权限验证，否则将不会生效

4、设置验证

能够登录成功 ，但是没有权限，说明前面的设置是成功了
