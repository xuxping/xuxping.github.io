mongo安装

CentOS7安装mongdb：
https://docs.mongodb.com/master/tutorial/install-mongodb-on-red-hat/

总共试过3种方法

  1. 官方镜像源，网速太慢，下载不下来
  2. 清华镜像源，无法使用
  3. aliyun的rpm包安装，安装成功

官方镜像源使用过程
创建文件/etc/yum.repos.d/mongodb-org-3.4.repo
文件内容为：

[mongodb-org-3.4]
name=MongoDB Repository
baseurl=https://repo.mongodb.org/yum/amazon/2013.03/mongodb-org/3.4/x86_64/
gpgcheck=1
enabled=1
gpgkey=https://www.mongodb.org/static/pgp/server-3.4.asc

使用yum进行安装

yum makecache
yum install mongodb-org


wget http://mirrors.aliyun.com/mongodb/yum/redhat/7/mongodb-org/3.4/x86_64/RPMS/mongodb-org-3.4.0-1.el7.x86_64.rpm 
 wget http://mirrors.aliyun.com/mongodb/yum/redhat/7/mongodb-org/3.4/x86_64/RPMS/mongodb-org-mongos-3.4.0-1.el7.x86_64.rpm 
wget http://mirrors.aliyun.com/mongodb/yum/redhat/7/mongodb-org/3.4/x86_64/RPMS/mongodb-org-server-3.4.0-1.el7.x86_64.rpm

新建一个/home/mongodb目录
vim /etc/mongod.conf
将
storage:
  dbPath: /var/lib/mongo/
改为：
storage:
  dbPath: /home/mongodb     

使配置生效
mongo —config /etc/mongod.conf


查看进程
ps -ef| grep mongo
//查看绑定的端口
netstat -tulnp | grep mongo
