脑电波数据采集

发表于 2017-01-08 | 分类于 project |

脑电波采集设备，戴在采集人的头上，通过蓝牙将数据传递到android软件中

采集的时长为5分钟，大概一个人能够采集到330条数据
收集到的数据格式：
time,highAlpha,lowAlpha,highBeta,lowBeta,delta,
lowGamma,midGamma,theta,attension，meditation

实现原理：利用NeruySky的ThinkGear包直接获取脑电数据，然后保存到文件中
主要的点在于安卓软件端，涉及到的知识点有，蓝牙的连接，Handler使用，文件的保存.
虽然我已远离android开发，但是还是能够快速玩转的
