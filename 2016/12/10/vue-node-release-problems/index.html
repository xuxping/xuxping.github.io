npm run build出现如下问题

遇到这种问题可能是node的版本问题。据查extract-text-webpack-plugin在node6.0.0版本上存在问题。

解决方法：  
    1.  更换node版本,从node6.0.0切换到7.0.0上。  
    2.  删除node_modules，然后重新装一遍。
